JogosDeEmpresa
==============
Sistema chamado JOGOS DE EMPRESA que tem como objetivo inserir no Curso de Administração uma solução tecnológica para uma matéria que era desenvolvida na prática pelos alunos. Possui as seguintes funcionalidades:
CADASTRO: Alunos, Cargo, Curso, Empresa, Alunos na Empresa, Jornal, Periodo, Produto, Termo.
MOVIMENTAÇÕES: Com as informações em mãos os alunos tomavam as decisões para um melhor desempenho das suas empresas num ciclo que ia até o fim do bimestre letivo.
